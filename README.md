# AutomatedVerificationProgram

## Project Description

This project is a **C# Windows Forms application** for automated verification.

It integrates:

* OCR (Optical Character Recognition)
* Automated / Manual verification processes
* Report generation
* System configuration settings

---

## Environment

* Visual Studio 2022 (or above)
* .NET Framework (check `.csproj`, usually 4.x)
* Windows OS

---

## How to Run

1. Clone or download this repository
2. Open `AutomatedVerificationProgram.sln` using Visual Studio
3. Restore NuGet packages (if needed)
4. Build the solution
5. Run the project

---

## Features

* Batch verification
* Manual verification
* OCR processing
* Report generation
* Custom verification sequence

---

## Notes

* `bin/` and `obj/` are excluded (auto-generated)
* `.vs/` is excluded (Visual Studio settings)
* If packages are missing:

  * Right click solution → Restore NuGet Packages
